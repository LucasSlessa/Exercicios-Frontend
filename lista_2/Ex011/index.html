Diferença entre <div> e <span>

A <div> é um elemento de bloco e é usada para agrupar elementos em um documento HTML e definir a estrutura da página. Ela geralmente é usada para criar seções ou áreas distintas em um layout.

Por outro lado, o <span> é um elemento de linha e é usado para aplicar estilos a partes de um texto sem quebrar a linha. Ele geralmente é usado para estilizar pequenas partes de texto ou para aplicar estilos em linha.
