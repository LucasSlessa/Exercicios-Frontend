O que é o http 🛡️🛡️

HTTP (Hypertext Transfer Protocol) é um protocolo de comunicação usado para transferir informações na World Wide Web.
Ele define a forma como as mensagens são formatadas e transmitidas, permitindo que os navegadores da web solicitem recursos, como páginas da web, imagens, vídeos, etc., de servidores web e exibam esses recursos aos usuário
