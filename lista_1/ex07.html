O que o xampp?

O XAMPP é um pacote de software gratuito que contém Apache, MySQL, PHP e Perl.
Ele é usado principalmente para criar um ambiente de desenvolvimento web local em sistemas Windows, macOS e Linux.
O XAMPP facilita a configuração de um servidor web local para testar e desenvolver sites e aplicativos web antes de implantá-los em um servidor de produção..
