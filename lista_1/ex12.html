HTML é uma linguagem de programação? Explique

HTML (Hypertext Markup Language) é a linguagem de marcação padrão usada para criar e estruturar páginas da web
Enquanto as linguagens de programação tradicionais, como Python, Java ou C++, são usadas para criar algoritmos e controlar o comportamento de um programa,
o HTML é usado para descrever a estrutura de uma página web e como o conteúdo deve ser apresentado.
